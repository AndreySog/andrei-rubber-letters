01 О сайте

// зачем

Публичная витрина идей Андрея. Каждая страница — самодостаточный документ: открывается на телефоне, шарится по ссылке, показывается друзьям / инвесторам / производству.

// форматы

«Нарисуй» — пиксельный трейс учебного образца. Визуально точно, не редактируется как чертёж.

«Начерти» — параметрические кривые по ЕСКД. Открывается в Illustrator / Inkscape / AutoCAD, пригодно для производства.

// стек

Чистый HTML + CSS + vanilla JS. Никаких бандлеров, фреймворков, CDN. Хостинг — GitHub Pages, деплой через git push.

// обновления

Каждая итерация коммитится в репозиторий. История версий — в git, актуальная версия — на этом сайте. Без рекламы, без трекинга.
