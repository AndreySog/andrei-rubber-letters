# Идеи Андрея — andrei-ideas-site

Интерактивный сайт-витрина идей и прототипов Андрея.

🌐 **Live:** https://AndreySog.github.io/andrei-ideas-site/

## Структура

```
.
├── index.html              — главная страница (лендинг)
├── ideas/
│   ├── mobius-unit.html    — Мёбиус-интегрированный узел
│   └── bolt-m8.html        — Болт М8 × 40 с переключателем нарисуй/начерти
└── assets/
    ├── css/main.css        — стили (тёмная тема)
    ├── js/main.js          — переключатель и UI-логика
    └── img/                — SVG-чертежи
```

## Что внутри

### Мёбиус-интегрированный узел
Лента Мёбиуса как несущая конструкция + катушки возбуждения + магниты. Хиральный EM-ответ, сниженная самоиндукция, не-реципрокные моды.

### Болт М8 × 40 (ГОСТ 7798-70)
Демонстрация двух форматов:
- **Нарисуй** — пиксельный трейс учебного образца (Рис. 51)
- **Начерти** — параметрический чертёж по ЕСКД

## Как добавить новую идею

1. Создать `ideas/<slug>.html` по образцу существующих.
2. Добавить карточку на `index.html` в секции `#projects`.
3. Закоммитить, запушить в `main`. GitHub Pages обновится автоматически.

## Хостинг

GitHub Pages, ветка `main`, корень репозитория.

## Защита паролем

Сайт закрыт паролем от случайных посетителей. Пароль — `qqq`.

**Это клиентский гейт (client-side gate)**, а не настоящая авторизация. Любой, кто посмотрит исходник страницы или `assets/js/lock.js`, увидит пароль и контент за ним. Это нормально для друзей-и-семьи как наш случай.

Если потребуется настоящая защита (rate-limit, аккаунты, защита от скрейпинга) — это отдельный backend (Cloudflare Access, Netlify Identity, Auth0, или свой на VPS).

**Логаут для разработчиков:** откройте devtools и выполните `lockOut()`.
